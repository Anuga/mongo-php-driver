\MongoDB\Query

\MongoDB\QueryQuery

Value object corresponding to a wire protocol OP_QUERY message.

If and when queries become commands, we will need to introduce a new Query object, such as QueryCommand. At that point, the query will likely be constructed from a single document, which includes the arguments below in a similar fashion to findAndModify.
