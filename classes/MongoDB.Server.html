\MongoDB

\MongoDBServer

Server abstracts a socket connection to a single MongoDB server. The server itself may be a mongod (stand-alone or replica set node) or mongos process.

Users will typically not construct this class directly. The common use case will be connection to a cluster of servers via a URI with the Manager class.

This class does not utilize read preferences, since there is only a single single socket on which to send a command, query, or write.

Operation methods do not take socket-level options (e.g. socketTimeoutMS). Those options should be specified during construction.
